Bluemix Quick Start

Create an IBM id

https://www.ibm.com/account/us-en/signup/register.html
Ignore if you already have IBM id.

Sign Up or Log in Bluemix

  1. Bluemix public https://console.ng.bluemix.net/
  2. IBM Bluemix DevOps Services https://hub.jazz.net/

Create application

  1. Click CREATE APP -> WEB -> PHP and fill one app name (need to be unique)
  2. Read start coding docuemnt (CF and GIT)
  3. Click ‘Overview’ in left navigation.
  4. Click ‘ADD GIT’ then ‘CONTINUE’ to create git repo. ‘ADD GIT’ is now replaced with GIT URL and EDIT CODE button.
  5. Copy ‘Routes’ url appname.mybluemix.net and open with browser

Use Jazz (Bluemix DevOps Services)

  1. Right click in left navigation and create a new file: info.php
  2. Put <?php phpinfo(); ?> in info.php
  3. Click ‘Git Repository’, select info.php and commit, then push the commit
  4. Go back to Edit page and click ‘Deploy’ icon (next to app status bar)
  5. Open http://appname.mybluemix.net/info.php in browser

Git local

  1. Check email from sent from IBM DevOps Services (hub@jazz.net) and get Git Repository URL: https://hub.jazz.net/git/username/appname
  2. git clone https://hub.jazz.net/git/username/appname
  3. Create a new file: test.php with <?php echo ‘test’; ?>
  4. git add .
  5. git commit -m 'add test.php'
  6. git push origin master
  7. Open http://appname.mybluemix.net/test.php in browser
