Install APCU on OSX (PHP5.5)

Hope that this article helps :)

Download and install

  • Dowload from http://pecl.php.net/package/APCu
  • Extract apcu-4.0.8.tgz
  • Run

If you meet “pcre.h” error on make test, then try

  • Run brew install pcre pcre++
  • Modify /apcu-4.0.8/Makefile, add -I/usr/local/include to INCLUDES =

/usr/local/include is where Homebrew would put the pcre.h file
